Bugs
- None recorded yet.
- Needs real remote ICS fetching instead of embedded seed ICS examples.
- Outlook sync is a scaffold only and does not yet implement Microsoft Graph OAuth.
- Frontend currently expects backend API availability and does not include offline mock fallback.
- Seed data originally used string datetimes and was corrected to real Python datetime objects for SQLite compatibility.
- Weekly scan automation currently uses app runtime scheduling, so scans occur only while the backend process is running.
- Shared password lock is suitable for MVP gating, but not a replacement for full user authentication.
- Add-to-calendar currently uses ICS export rather than direct Outlook or Google write APIs.
- Scan progress bar is a frontend progress experience and does not yet stream backend per-source progress updates.
- Scan endpoint previously returned `{}` and was updated to return JSON-safe run payloads.
- Event deletion was missing from the UI and API and is now implemented.
- Source deletion and saved-item removal were missing from the management flows and are now implemented.
